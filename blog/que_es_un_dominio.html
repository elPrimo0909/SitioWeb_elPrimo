¿Qué es un dominio?

05 Enero 2023

El dominio es uno de los elementos más cruciales de cualquier página web, o mejor dicho de todo el Internet. Esta parte aparentemente pequeña de la red mundial es en realidad la razón por la que puedes visitar tus tiendas online y blogs favoritos. Sin él, navegar por la red se sentiría mucho más como una clase de álgebra y construir una presencia en línea sería un reto casi imposible.
Si planeas abrir una tienda online o compartir tu pasión con el mundo, entonces necesitarás saber qué es un dominio web. Tu nombre de dominio no solo afectará la probabilidad de que las personas encuentren tu sitio web, sino que también desempeñará un papel importante en la forma en que las personas lo perciben desde el principio, incluso antes de que vean lo que tienes para ofrecer.

¿Qué es un dominio?
Un nombre de dominio es la dirección que las personas escriben en un navegador para llegar a un sitio específico. Al igual que una huella digital, cada nombre de dominio es exclusivo de un sitio web y no se puede compartir ni duplicar con otras páginas web. Si observas la parte superior de esta página, verás que el nombre de dominio de este sitio web es "amonca.com".
Los dominios se crearon como una forma sencilla de acceder al Protocolo de Internet (IP), que representa el localizador en línea de un sitio web. Una dirección IP es una cadena de números asignados a cada ordenador, que consta de cuatro números decimales que van del 0 al 255 separados por puntos. Si bien estas series aparentemente aleatorias de números son excelentes para las computadoras, sin embargo para los humanos es mucho más fácil usar palabras que podemos recordar. Y similar a guardar un número en tu teléfono, los nombres de dominio nos permiten guardar "172.217.3.196" como "google.com".
Los registros de nombres de dominio son supervisados ​​por la Corporación de Internet para la Asignación de Nombres y Números (ICANN), que asigna direcciones IP, ejecuta sistemas de acreditación para registradores de dominios y mantiene una base de datos centralizada de todos los nombres de dominio y sus IP. Como veremos en más detalle más adelante en esta guía, la ICANN también tiene la autoridad para aprobar nuevas extensiones de dominio (también conocidas como TLD), administrarlas y cerrarlas si no siguen las reglas determinadas.

¿Cómo funcionan un dominio?
Internet es una red mundial masiva de computadoras conectadas entre sí a través de una red submarina global de cables. Cada uno de estos equipos, independientemente de si funcionan como dispositivos personales o servidores, se identifica mediante una dirección IP que les permite comunicarse entre sí para enviar, buscar y recuperar datos web.
Los nombres de dominio son simplemente versiones fáciles de usar de estas direcciones IP, conectadas a sitios web específicos. Sin embargo, para que las computadoras puedan encontrar las páginas web correctas, las cadenas de palabras aparentemente aleatorias deben volver a convertirse en números. Aquí es donde entra en juego el sistema de nombres de dominio.
Conocido simplemente como DNS, el sistema de nombres de dominio traduce cualquier nombre de dominio ingresado en un navegador a una IP legible por computadora. El sistema es supervisado por la ICANN y comúnmente se compara con las guías telefónicas de la vieja escuela en las que buscarías el nombre de cualquier persona para encontrar su número. Si eres demasiado joven para entender esta referencia, considerarla como una lista de contactos en un teléfono que contiene números con los que nunca has interactuado.
Al ingresar un nombre de dominio en un navegador web, se envía una solicitud a los servidores DNS que buscan los servidores de nombres conectados a ese dominio específico y les envían tu solicitud. Estos servidores de nombres son administrados por el proveedor de alojamiento web y, después de encontrar la IP relevante, enviarán la solicitud al servidor web donde se almacenan los archivos del sitio web. El servidor web usa la IP para encontrar todos los archivos asociados con él y enviar todos los datos al navegador. Todo esto sucede en menos de 3 segundos.

La estructura de un dominio
Un nombre de dominio completo se conoce como FQDN, por sus siglas en inglés. Un nombre de dominio completo o FQDN consiste de mínimo tres niveles:
Dominio de nivel superior (TLD)
Dominio de segundo nivel (SLD)
Dominio de tercer nivel
Los niveles en un nombre de dominio completo están estructurados en una jerarquía de árbol, separados entre sí por un punto, y se leen siempre de derecha a izquierda. Primero se identifica un dominio en base a su dominio de nivel superior (TLD) o extensión, después el dominio de segundo nivel (SLD) o dominio (la parte del dominio que se registra) y por último el dominio de tercer nivel o nombre del host.
Por ejemplo en el caso de "amonca.com": "www" es el dominio de tercer nivel, "amonca" es el dominio de segundo nivel y ".com" es el dominio de nivel superior. Donde cada elemento está separado por un punto y es leído de derecha a izquierda.

¿Cómo elegir un nombre de dominio?
Después de haber aprendido acerca de la estructura y niveles de un dominio completo, tal vez quieras elegir tu propio nombre de dominio.
Para encontrar un nombre de dominio que represente perfectamente tu marca o concepto y que aún no se haya tomado, necesitarás una combinación de creatividad, conocimiento de SEO, previsión, suerte y un amplio conocimiento sobre las mejores prácticas a seguir. Para asegurarnos de cubrir todo esto, hemos elaborado una guía completa sobre cómo elegir el mejor nombre de dominio. Pero por le momento, estos son tres de los pasos más importantes para comenzar:

1. Que sea fácil de escribir y pronunciar:
Si tienes un nombre poco común, o conoces a alguien que lo tiene, probablemente te has dado cuenta de cuántas personas lo escriben mal o peor aún lo olvidan por completo. Debido a esto, es crucial encontrar un nombre de dominio que sea fácil de escribir y pronunciar.
Al comenzar a pensar en ideas para nombres de dominio, querrás evitar palabras que se pronuncian mal con frecuencia o que sean difíciles de escribir. Además, recomendamos no usar abreviaturas y palabras que se puedan escribir de muchas maneras.
Cuando te sientas satisfecho con las ideas que tienes, es hora de ponerlas a prueba. Checa tus opciones preferidas con algunos amigos para ver si pueden escribirlo y recordarlo con facilidad. Si tienen dificultades, elige otra opción.

2. Mantén el nombre de dominio acorde a tu marca:
Tu nombre de dominio debe reflejar tu marca. Es ideal usar el mismo nombre de tu marca, más una palabra clave para optimizar su capacidad de búsqueda. Muchos empresarios tienden a pasar por alto el aspecto de su marca en el nombre de dominio de su sitio y después cambiarlo puede generar dolores de cabeza innecesarios e incluso la pérdida de clientes.

3. Elige la extensión de dominio correcta:
Cuando buscas la mejor extensión para tu nombre de dominio, hay dos cosas principales que debes tener en cuenta: la percepción y el precio. Además de respetar las reglas de uso, también deberás tener en cuenta de que las personas confían en ciertos TLDs y los recuerden.
